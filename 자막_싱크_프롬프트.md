# 자막 싱크 기능 - API 및 구현 가이드

## 📋 개요
TTS로 생성된 오디오와 자막의 타이밍을 정확하게 맞추는 기능입니다. OpenAI Whisper API를 사용하여 단어별 정확한 타이밍을 추출하고, 프레임 정렬 및 겹침 방지 로직을 통해 완벽한 자막 싱크를 구현합니다.

---

## 🔌 API 정보

### 1. Whisper Align API (단어별 타이밍 추출)

#### 기본 정보
- **API 엔드포인트**: `POST /api/whisper-align`
- **HTTP 메서드**: `POST`
- **Content-Type**: `multipart/form-data`
- **사용 모델**: OpenAI Whisper (`whisper-1`)

#### 요청 형식

```typescript
const formData = new FormData()
formData.append("audio", audioBlob, "audio.wav")
formData.append("text", "정답 대본 텍스트")
formData.append("apiKey", "openai_api_key") // 선택사항

const response = await fetch("/api/whisper-align", {
  method: "POST",
  body: formData,
})
```

#### 요청 파라미터

- **audio** (File, 필수): 분석할 오디오 파일 (WAV 형식 권장)
- **text** (string, 필수): 정답 대본 텍스트 (Forced Alignment용)
- **apiKey** (string, 선택): OpenAI API 키 (없으면 환경 변수 사용)

#### OpenAI Whisper API 호출

#### Forced Alignment 로직

1. **정답 대본 단어 분리**
   - 공백과 문장부호를 모두 포함
   - 문장부호만 있는 단어는 이전 단어와 결합
   - 한 글자 단어는 이전 단어와 결합 (자막 분할 시 한 글자만 나오는 문제 방지)

2. **Whisper 결과와 정답 대본 매칭**
   - 정답 대본의 단어 순서를 기준으로 Whisper 결과를 정렬
   - 각 정답 단어에 대해 가장 가까운 Whisper 결과를 찾아 매칭
   - 유사도 계산: 정확히 일치 > 부분 일치 (70% 이상)
   - 매칭 실패 시 이전 단어의 시간을 기반으로 추정

3. **단어 유사도 계산**

```typescript
function calculateWordSimilarity(word1: string, word2: string): number {
  if (word1 === word2) return 1.0
  if (word1.includes(word2) || word2.includes(word1)) return 0.8
  
  // 간단한 편집 거리 계산
  const len1 = word1.length
  const len2 = word2.length
  const maxLen = Math.max(len1, len2)
  if (maxLen === 0) return 1.0
  
  let matches = 0
  const minLen = Math.min(len1, len2)
  for (let i = 0; i < minLen; i++) {
    if (word1[i] === word2[i]) matches++
  }
  
  return matches / maxLen
}
```

## 📋 자막 타이밍 조정 로직

### 1. 프레임 정렬

영상 프레임레이트에 맞춰 자막 타이밍을 정렬하여 싱크 오차를 원천 차단합니다.

```typescript
const VIDEO_FPS = 25 // 영상 프레임레이트
const FRAME_DURATION = 1 / VIDEO_FPS // 0.04초 (25fps 기준)
const SUBTITLE_SAFETY_MARGIN = 0.02 // 0.02초 안전 마진

function alignToFrame(time: number): number {
  const frames = Math.round(time / FRAME_DURATION)
  return frames * FRAME_DURATION
}

// 모든 자막의 start/end 시간을 프레임 단위로 정렬
for (let i = 0; i < subtitles.length; i++) {
  subtitles[i].start = Number.parseFloat(alignToFrame(subtitles[i].start).toFixed(10))
  subtitles[i].end = Number.parseFloat(alignToFrame(subtitles[i].end).toFixed(10))
}
```

### 2. 자막 겹침 방지

자막 간 겹침을 방지하고 간격을 채워 연속적으로 표시합니다.

```typescript
// 1. start 시간 순으로 정렬
subtitles.sort((a, b) => a.start - b.start)

// 2. 자막 endTime 확장: 각 자막의 endTime을 다음 자막의 startTime까지 무조건 확장
for (let i = 0; i < subtitles.length - 1; i++) {
  const currentSubtitle = subtitles[i]
  const nextSubtitle = subtitles[i + 1]
  
  // 타이밍 겹침 방지: 다음 자막의 startTime이 이전 자막의 endTime보다 작거나 같으면 조정
  if (nextSubtitle.start <= currentSubtitle.end) {
    // 다음 자막의 startTime을 이전 자막의 endTime + 1프레임(0.04초)으로 설정
    nextSubtitle.start = Number.parseFloat(alignToFrame(currentSubtitle.end + FRAME_DURATION).toFixed(10))
  }
  
  // 간격이 있으면 현재 자막의 endTime을 다음 자막의 startTime까지 무조건 확장
  currentSubtitle.end = Number.parseFloat(nextSubtitle.start.toFixed(10))
}

// 3. 모든 자막의 endTime이 startTime보다 큰지 확인
for (let i = 0; i < subtitles.length; i++) {
  if (subtitles[i].end <= subtitles[i].start) {
    // endTime이 startTime보다 작거나 같으면 최소 1프레임(0.04초) 길이로 설정
    subtitles[i].end = Number.parseFloat((subtitles[i].start + FRAME_DURATION).toFixed(10))
  }
}
```

### 3. 실제 오디오 길이에 맞춘 조정

실제 오디오 길이와 요청된 duration이 다르면 자막 타이밍을 비율로 조정합니다.

```typescript
// 실제 오디오 길이 측정 (ffprobe 사용)
const probe_cmd = [
  'ffprobe',
  '-v', 'quiet',
  '-print_format', 'json',
  '-show_format',
  audio_path
]
const probe_result = subprocess.run(probe_cmd, capture_output=True, text=True)
const probe_data = json.loads(probe_result.stdout)
const actual_audio_duration = float(probe_data.get('format', {}).get('duration', 0))

// 자막 타이밍 조정
if (actual_audio_duration > 0 && duration > 0) {
  const duration_ratio = actual_audio_duration / duration
  
  for (const subtitle of subtitles) {
    subtitle.start = subtitle.start * duration_ratio
    subtitle.end = subtitle.end * duration_ratio
  }
}
```

## 📋 자막 생성 프로세스

### 1. 의미 단위별 STT 분석

각 문장을 의미 단위(phrase)로 나누어 개별 STT 분석을 수행합니다.

```typescript
// 1. 문장을 의미 단위로 분할
const phrases = sentence.split(/[,，.。!！?？;；:：]/).filter(p => p.trim().length > 0)

// 2. 각 의미 단위별로 개별 STT 분석
const phraseSttTasks: Array<{
  index: number
  phraseInfo: {
    lineId: string
    phrase: string
    startTime: number
    duration: number
  }
  lineStartTime: number
  lineAudioBuffer: AudioBuffer
}> = []

for (let i = 0; i < phrases.length; i++) {
  const phrase = phrases[i].trim()
  if (phrase.length === 0) continue
  
  // 해당 의미 단위의 오디오 구간 추출
  const relativeStartTime = phraseStartTime - lineStartTime
  const relativeEndTime = relativeStartTime + phraseDuration
  const segmentBuffer = extractAudioSegment(
    lineAudioBuffer,
    Math.max(0, relativeStartTime),
    Math.min(lineAudioBuffer.duration, relativeEndTime)
  )
  
  phraseSttTasks.push({
    index: i,
    phraseInfo: {
      lineId: lineId,
      phrase: phrase,
      startTime: phraseStartTime,
      duration: phraseDuration
    },
    lineStartTime: lineStartTime,
    lineAudioBuffer: lineAudioBuffer
  })
}

// 3. 배치 병렬 처리 (5개씩)
const BATCH_SIZE = 5
const totalBatches = Math.ceil(phraseSttTasks.length / BATCH_SIZE)

for (let batchStart = 0; batchStart < phraseSttTasks.length; batchStart += BATCH_SIZE) {
  const batch = phraseSttTasks.slice(batchStart, batchStart + BATCH_SIZE)
  
  // 배치 내 병렬 처리
  const batchResults = await Promise.all(
    batch.map(async (task) => {
      // 오디오 구간 추출
      const segmentBuffer = extractAudioSegment(...)
      const audioBlob = await audioBufferToBlob(segmentBuffer)
      
      // Whisper align API 호출
      const formData = new FormData()
      formData.append("audio", audioBlob, `phrase_${task.phraseInfo.lineId}_${task.index}.wav`)
      formData.append("text", task.phraseInfo.phrase)
      
      const whisperResponse = await fetch("/api/whisper-align", {
        method: "POST",
        body: formData,
      })
      
      const whisperData = await whisperResponse.json()
      return {
        index: task.index,
        phraseInfo: task.phraseInfo,
        wordTimings: whisperData.wordTimings || [],
        success: true,
      }
    })
  )
  
  // 결과를 절대 시간으로 변환
  for (const result of batchResults) {
    if (result.success && result.wordTimings && result.wordTimings.length > 0) {
      const phraseStartPrecise = Number.parseFloat(result.phraseInfo.startTime.toFixed(10))
      
      for (const wordTiming of result.wordTimings) {
        const absoluteStart = Number.parseFloat((phraseStartPrecise + wordTiming.start).toFixed(10))
        const absoluteEnd = Number.parseFloat((phraseStartPrecise + wordTiming.end).toFixed(10))
        
        allWordTimings.push({
          word: wordTiming.word.trim(),
          start: absoluteStart,
          end: absoluteEnd,
          lineId: result.phraseInfo.lineId,
        })
      }
    }
  }
}
```

### 2. 자막 라인 생성

단어 타이밍을 기반으로 15~20자 단위로 자막 라인을 생성합니다.

```typescript
// 텍스트를 15~20자씩 나누는 함수
const splitTextForSubtitles = (text: string, minChars: number = 15, maxChars: number = 20): Array<{ text: string; startChar: number; endChar: number }> => {
  const chunks: Array<{ text: string; startChar: number; endChar: number }> = []
  let currentChunk = ""
  let startChar = 0
  
  // 공백과 구두점을 기준으로 단어 단위로 분할
  const words = text.split(/(\s+)/)
  
  for (let i = 0; i < words.length; i++) {
    const word = words[i]
    const testChunk = currentChunk + word
    
    // 최소 길이 이상이고 최대 길이를 초과하면 현재 청크 저장
    if (testChunk.length >= minChars && testChunk.length > maxChars && currentChunk.length >= minChars) {
      chunks.push({
        text: currentChunk.trim(),
        startChar: startChar,
        endChar: startChar + currentChunk.length
      })
      startChar += currentChunk.length
      currentChunk = word
    } else {
      currentChunk = testChunk
    }
    
    // 구두점(., !, ?)이 있으면 강제로 나누기
    if (/[.!?。！？]$/.test(word) && currentChunk.length >= minChars) {
      chunks.push({
        text: currentChunk.trim(),
        startChar: startChar,
        endChar: startChar + currentChunk.length
      })
      startChar += currentChunk.length
      currentChunk = ""
    }
  }
  
  // 마지막 청크 추가
  if (currentChunk.trim().length > 0) {
    chunks.push({
      text: currentChunk.trim(),
      startChar: startChar,
      endChar: startChar + currentChunk.length
    })
  }
  
  return chunks
}

// 자막 라인 생성
const subtitleLines = splitTextForSubtitles(sentence)
const wordTimingsForLine = allWordTimings.filter(wt => wt.lineId === lineId)

for (let j = 0; j < subtitleLines.length; j++) {
  const subtitleText = subtitleLines[j]
  const startChar = subtitleText.startChar
  const endChar = subtitleText.endChar
  
  // 해당 자막 라인에 해당하는 단어들 찾기
  let lineStartTime = 0
  let lineEndTime = 0
  let charCount = 0
  let foundStart = false
  
  for (const wordTiming of wordTimingsForLine) {
    const wordLength = wordTiming.word.length
    const wordStartChar = charCount
    const wordEndChar = charCount + wordLength
    
    // 자막 라인 범위에 포함되는 단어
    if (wordStartChar < endChar && wordEndChar > startChar) {
      if (!foundStart) {
        lineStartTime = wordTiming.start
        foundStart = true
      }
      lineEndTime = wordTiming.end
    }
    
    charCount += wordLength + 1 // 공백 포함
  }
  
  if (foundStart && lineEndTime > lineStartTime) {
    subtitles.push({
      id: `${lineId}_${j}`,
      start: Number.parseFloat(lineStartTime.toFixed(10)),
      end: Number.parseFloat(lineEndTime.toFixed(10)),
      text: subtitleText.text
    })
  }
}
```

### 3. 침묵 구간 감지 및 타이밍 조정

0.2초 이상의 침묵 구간을 감지하여 자막 타이밍을 조정합니다.

```typescript
// 침묵 구간 감지 (0.2초 이상)
for (let i = 0; i < allWordTimings.length - 1; i++) {
  const currentWord = allWordTimings[i]
  const nextWord = allWordTimings[i + 1]

    // 침묵 구간 감지: 다음 단어의 startTime을 현재 단어의 endTime + 0.1초로 조정
    const correctedStart = currentWord.end + 0.1

  }
}
```

### 2. 세그먼트별 자막 처리

```python
for sub_idx, sub in enumerate(subtitles):

        next_sub_start = float(next_sub.get('start', next_sub.get('startTime', 0)))
        
        # 겹침 방지
        if sub_end > next_sub_start:
            sub_end = next_sub_start - 0.01  # 0.01초 여유
        else:
            # 확장: 다음 자막의 startTime까지
            sub_end = next_sub_start
    
    # 마지막 자막인 경우, 실제 오디오 길이 + 3초 여유에 맞춰 endTime 조정
    if is_last_segment and is_last_subtitle and actual_audio_duration > 0:

    elif adjusted_end - adjusted_start < min_duration:
        adjusted_end = adjusted_start + min_duration
```

## 📋 재시도 로직

Whisper API 호출 실패 시 최대 10회까지 재시도합니다.

```typescript
const MAX_RETRIES = 10
const RETRY_DELAY = 2000 // 2초

for (let attempt = 1; attempt <= MAX_RETRIES; attempt++) {
  try {
    const whisperResponse = await fetch("/api/whisper-align", {
      method: "POST",
      body: formData,
    })
    
    if (whisperResponse.ok) {
      const whisperData = await whisperResponse.json()
      if (whisperData.wordTimings && whisperData.wordTimings.length > 0) {
        return {
          wordTimings: whisperData.wordTimings,
          success: true,
        }
      }
    }
    
    // 실패 시 재시도
    if (attempt < MAX_RETRIES) {
      await new Promise(resolve => setTimeout(resolve, RETRY_DELAY))
      continue
    }
  } catch (error) {
    if (attempt < MAX_RETRIES) {
      await new Promise(resolve => setTimeout(resolve, RETRY_DELAY))
      continue
    }
  }
}
```
