# 대본 기획 - 자동 타입 선택 프롬프트

## 📋 개요
사용자가 콘텐츠 타입(Type A, B, C, D)을 선택하지 않았을 때, AI가 주제를 분석하여 가장 적합한 타입을 자동으로 선택하는 프롬프트입니다.

---

## 🔌 API 정보

### 기본 정보
- **API 엔드포인트**: Google Gemini API
- **HTTP 메서드**: `POST`
- **인증 방식**: API Key
- **사용 모델**: `gemini-2.0-flash-exp`

- 타깃: 한국인 성인 시청자
- 톤: 친근하지만 가볍지 않음, 이야기꾼의 구어체, 몰입감 최우선

1) 이탈 방지 최우선

- 추상적인 설명만 나열하지 말고, 장면·상황·사람이 떠오르도록 구성

4) 과도한 정보 나열 금지
- 사실은 이야기 속에 녹여서 배치
- "설명"보다 "상황 → 의미" 구조를 우선

────────────────────────
[기획 산출물 형식]
────────────────────────

1) 콘텐츠 타입
- [타입이 여기에 들어갑니다]

2) 전체 이야기 한 줄 요약
- 이 영상이 "결국 어떤 이야기를 하려는지"를 한 문장으로

3) 오프닝 기획
- 초반 훅의 핵심 질문 또는 충격 포인트

- 각 파트마다:

  · 감정 방향 (불안 / 긴장 / 충격 / 공감 / 희망 등)
  · 장면화 가능한 포인트 (사람, 상황, 공간, 상징 오브젝트)

5) 클라이맥스 / 핵심 메시지
- 이야기의 정점
- 시청자의 감정이 가장 크게 움직여야 하는 지점

6) 아웃트로 설계
- 오늘 이야기가 '지금 우리에게 왜 중요한지'
- 여운을 남기는 마무리 방향
- 다음 영상으로 자연스럽게 이어지는 문장(기획 관점)

[출력 제한]
────────────────────────

- 실제 내레이션 문장, 대사, TTS용 문구는 절대 포함하지 마십시오.
- 설명은 기획자의 시점에서 서술하십시오.
- 제작자 노트, 체크리스트, 파트 구분, 글자수 등은 절대 포함하지 마십시오.
- 오직 기획안만 출력하십시오. 추가 설명이나 메타 정보는 포함하지 마십시오.

### 자동 타입 선택 추가 프롬프트

```

[콘텐츠 구조 선택]
────────────────────────
아래 4가지 중, 주제에 가장 적합한 구조를 **하나만 선택**하여 기획하시오.

● Type A. 단일 서사 심층형
- 하나의 사건, 인물, 역사적 이야기
- 기승전결 + 감정 곡선 중심

● Type B. 컴필레이션 / 목록형
- 하나의 대주제 아래 3~5개의 사례
- 각 사례는 '미니 스토리' 구조

● Type C. 다각도 탐구형
- 하나의 대상을 여러 관점으로 분해
- 논리적 흐름 + 점층적 몰입

● Type D. 이야기형 (옛날이야기/민담)
- "옛날 옛날에..." 형식의 전래동화/민담 형식
- 등장인물 중심 구성
- 사건 전개 구조 (시작 → 사건 → 갈등 → 해결 → 교훈)

선택한 타입을 명확히 밝히고 기획을 진행할 것.

기획안의 "1) 콘텐츠 타입" 부분에는 선택한 타입을 명확히 표시하세요.
```

## 📝 User Prompt

```
[기획할 주제]: {topic}

위 주제에 가장 적합한 콘텐츠 타입을 선택하고, 그 타입에 맞게 기획안을 작성하세요.

주제를 분석하여 가장 적합한 타입을 선택하세요:
- Type A: 하나의 사건이나 인물을 깊이 있게 다루는 경우
- Type B: 여러 사례를 나열하는 경우
- Type C: 하나의 대상을 여러 관점으로 분석하는 경우
- Type D: 옛날이야기/민담 형식으로 전달하는 경우

선택한 타입을 "1) 콘텐츠 타입" 부분에 명확히 표시하세요.
```

## 📋 타입별 선택 기준

### Type A (단일 서사 심층형)
- **적합한 경우**:
  - 하나의 역사적 사건을 깊이 있게 다루는 주제
  - 특정 인물의 생애나 업적을 중심으로 하는 주제
  - 시간순으로 전개되는 하나의 이야기
  - 기승전결 구조가 명확한 주제

- **예시**: "세종대왕의 한글 창제", "임진왜란의 전개 과정", "이순신 장군의 생애"

### Type B (컴필레이션 / 목록형)
- **적합한 경우**:
  - 하나의 대주제 아래 여러 사례를 나열하는 주제
  - "역사상 가장 ~한 사건들", "~한 인물들" 같은 주제
  - 각 사례가 독립적인 미니 스토리로 구성 가능한 주제

- **예시**: "역사상 가장 치열했던 전투 5가지", "조선시대 명장 3인", "한국사 최고의 승리"

### Type C (다각도 탐구형)
- **적합한 경우**:
  - 하나의 대상을 여러 관점으로 분석하는 주제
  - 역사적, 사회적, 경제적, 문화적 관점으로 분해 가능한 주제
  - 논리적 흐름과 점층적 몰입이 필요한 주제

- **예시**: "조선시대 양반제도의 다각도 분석", "임진왜란의 원인과 결과", "한국의 경제 발전 과정"

### Type D (이야기형)
- **적합한 경우**:
  - 전래동화나 민담 형식으로 전달하기 적합한 주제
  - 등장인물 중심의 이야기 구조
  - "옛날 옛날에..." 형식으로 시작할 수 있는 주제
  - 교훈이나 의미를 자연스럽게 전달할 수 있는 주제

- **예시**: "옛날 옛날에 세종대왕이...", "한 옛날에 이순신 장군이...", "옛날 이야기: 조선의 명장들"

## ⚠️ 주의사항

1. **타입 선택 필수**: AI가 주제를 분석하여 4가지 타입 중 하나를 반드시 선택해야 합니다.
2. **명확한 표시**: 선택한 타입을 "1) 콘텐츠 타입" 부분에 명확히 표시해야 합니다.
3. **타입별 구조 준수**: 선택한 타입의 구조와 특징에 맞게 기획안을 작성해야 합니다.
4. **단일 타입 선택**: 여러 타입을 혼합하지 않고 하나의 타입만 선택해야 합니다.
