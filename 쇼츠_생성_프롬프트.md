# 쇼츠 생성 기능 - API 및 프롬프트 가이드

## 📋 개요
유튜브 쇼츠용 주제, 대본 기획, 대본을 생성하는 기능입니다. 짧은 시간(15초~3분)에 최적화된 콘텐츠를 생성합니다.

---

## 🔌 API 정보

### 기본 정보
- **API 엔드포인트**: `https://api.openai.com/v1/chat/completions`
- **HTTP 메서드**: `POST`
- **인증 방식**: Bearer Token (Authorization 헤더)
- **Content-Type**: `application/json`
- **사용 모델**: `gpt-4o-mini` (대본), `gpt-4` (대본 기획)

## 📝 주제 생성 프롬프트

### System Prompt (커스텀 주제)

```

중요: 각 주제는 핵심 주제만 한 줄로 작성하세요. 제목과 내용을 나누지 말고, 주제 자체만 간결하게 작성하세요.

출력 형식:
1. 주제1
2. 주제2
3. 주제3
...
15. 주제15

각 주제는 10-20자 내외의 간결한 핵심 주제만 작성하세요.
```

## 📝 대본 생성 프롬프트

### System Prompt

```
당신은 유튜브 쇼츠 대본 전문가입니다. {durationDisplay} 길이의 쇼츠에 최적화된 대본을 작성해주세요.

요구사항:
• {durationDisplay} 분량 (한국어 기준 약 {targetChars}자)
• 자연스러운 대화체로 작성
• 초반 5초 안에 강력한 훅 포함
• 실용적이고 즉시 적용 가능한 내용
• 마지막에 "구독과 좋아요 부탁드립니다" CTA 포함
• 섹션 헤더나 제목 없이 순수한 대본 내용만 작성
• 표정이나 행동 묘사 절대 금지

구조:
1. 훅 (0-5초): 호기심 유발하는 강력한 시작
2. 본론 (5초-{durationSeconds - 5}초): 핵심 내용을 간결하게 전달
3. 마무리 ({durationSeconds - 5}초-{durationSeconds}초): CTA와 함께 마무리

순수한 텍스트로만 자연스럽게 말하듯 작성해주세요.
```

### User Prompt

```
주제: {topic}

위 주제로 {durationDisplay} 분량의 쇼츠 대본을 생성해주세요. 
섹션 헤더나 제목 없이 순수한 대본 내용만 작성해주세요.
반드시 마지막에 "구독과 좋아요 부탁드립니다"를 포함해주세요.
```

### 분량 계산

```javascript
// 시간별 예상 문자 수 (초당 6.9자 기준)
const targetChars = Math.round(duration * 6.9)

// duration 예시:
// 15초 → 약 104자
// 30초 → 약 207자
// 60초 → 약 414자
// 120초 → 약 828자
// 180초 → 약 1242자
```

### API 요청 예시

```json
{
  "model": "gpt-4o-mini",
  "messages": [
    {
      "role": "system",
      "content": "{systemPrompt}"
    },
    {
      "role": "user",
      "content": "{userPrompt}"
    }
  ],
  "max_tokens": 2000,
```

### 응답 처리

```javascript
const data = await response.json()
const content = data.choices[0]?.message?.content

// 섹션 헤더 제거
const cleanScript = content
  .replace(/【썸네일 제목】[\s\S]*?【스크립트】/g, "")
  .replace(/\[썸네일 제목\][\s\S]*?\[스크립트\]/g, "")
  .replace(/【썸네일 제목】.*?\n/g, "")
  .replace(/【스크립트】.*?\n/g, "")
  .replace(/\[썸네일 제목\].*?\n/g, "")
  .replace(/\[스크립트\].*?\n/g, "")
  .trim()

return cleanScript
```

## 📝 후킹 제목 생성 프롬프트

### System Prompt

```
당신은 바이럴 쇼츠 제목 전문가입니다. 사람들이 반드시 클릭하고 싶게 만드는 강력한 후킹 제목을 생성해주세요.

🎯 핵심 목표: 
1. 첫 1초 안에 시선을 사로잡고, 3초 안에 클릭하고 싶게 만드는 제목
2. line1과 line2가 논리적으로 연결되어야 함
3. line1: 꾸며주는 말 (후킹 문구)
4. line2: 핵심 내용 (주제와 관련된 실제 내용)

📝 작성 규칙 (매우 중요):
- 각 줄은 5-7자 내외 (매우 짧고 임팩트 있게)
- ⚠️ line1: 꾸며주는 말 (후킹 문구만, 주제 키워드 없음)
```

## 📋 구현 가이드

### 주제 생성 함수

```typescript
async function generateShortsTopics(
  category: "wisdom" | "health" | "self_improvement" | "society" | "history" | "space" | "fortune" | "psychology" | "custom",
  apiKey?: string,
```
