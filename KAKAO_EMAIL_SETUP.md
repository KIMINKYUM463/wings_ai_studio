# 카카오 로그인 이메일 수집 설정 가이드

## 문제 상황
카카오 개발자 센터에서 이메일 동의 항목이 "권한 없음" 상태로 표시되어, 이메일을 수집할 수 없습니다.

## 해결 방법

### 1단계: 카카오 개발자 센터 접속
1. https://developers.kakao.com/ 접속
2. 로그인 후 **내 애플리케이션** 클릭
3. 해당 애플리케이션 선택

### 2단계: 카카오 로그인 설정
1. 왼쪽 메뉴에서 **카카오 로그인** 클릭
2. **활성화 설정** 탭에서 **카카오 로그인 활성화** 확인

### 3단계: 동의 항목 설정
1. 왼쪽 메뉴에서 **카카오 로그인** > **동의 항목** 클릭
2. **개인정보** 섹션에서 **카카오계정(이메일)** 찾기
3. **설정** 버튼 클릭 (현재 "권한 없음" 상태)

### 4단계: 이메일 동의 항목 활성화
1. **동의 목적** 입력 (예: "서비스 이용 및 고객 지원")
2. **필수 동의** 또는 **선택 동의** 선택
   - **필수 동의**: 로그인 시 반드시 동의해야 함
   - **선택 동의**: 사용자가 선택적으로 동의 가능
3. **저장** 버튼 클릭

### 5단계: 코드에 이메일 scope 추가
동의 항목 활성화 후, `app/api/kakao/auth/route.ts` 파일에서:

```typescript
// scope에 account_email 추가
const scope = "account_email,profile_nickname,profile_image"
```

### 6단계: 테스트
1. 카카오 로그인 다시 시도
2. 이메일 동의 화면 표시 확인
3. Supabase `users` 테이블에서 이메일 수집 확인

## 주의사항

⚠️ **이메일 동의 항목 활성화는 심사가 필요할 수 있습니다**
- 카카오에서 서비스 심사를 통해 승인해야 할 수 있음
- 심사 기간: 보통 1-3일 소요

⚠️ **전화번호는 카카오 로그인으로 수집 불가능**
- 전화번호 수집을 위해서는 **카카오싱크** 사용 필요
- 카카오싱크는 별도 신청 및 심사 필요

## 현재 상태
- ✅ 닉네임: 수집 가능 (이미 활성화됨)
- ✅ 프로필 이미지: 수집 가능 (이미 활성화됨)
- ⏳ 이메일: 동의 항목 활성화 필요
- ❌ 전화번호: 카카오 로그인으로 불가능 (카카오싱크 필요)
