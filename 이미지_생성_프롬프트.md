# 롱폼 이미지 생성 기능 - API 및 프롬프트 가이드

## 📋 개요
대본 텍스트를 분석하여 이미지 생성용 영어 프롬프트를 생성하고, Replicate API를 통해 실제 이미지를 생성하는 기능입니다.

---

## 🔌 API 정보

### 1. 이미지 프롬프트 생성 (OpenAI)

#### 기본 정보
- **API 엔드포인트**: `https://api.openai.com/v1/chat/completions`
- **HTTP 메서드**: `POST`
- **인증 방식**: Bearer Token (Authorization 헤더)
- **Content-Type**: `application/json`
- **사용 모델**: `gpt-4o-mini`

#### 파라미터
- `max_tokens`: 500
- `temperature`: 0.7

### 2. 이미지 생성 (Replicate)

#### 기본 정보
- **API 엔드포인트**: `https://api.replicate.com/v1/models/{model}/predictions`
- **HTTP 메서드**: `POST`
- **인증 방식**: Token (Authorization 헤더)
- **Content-Type**: `application/json`

#### 지원 모델
- `black-forest-labs/flux-pro` (기본, 고품질)
- `black-forest-labs/flux-schnell` (빠른 생성)
- `prunaai/hidream-l1-fast` (스틱맨 애니메이션)
- `google/imagen-4-fast` (빠른 생성, 16:9 전용)
- `stability-ai/sdxl` (fallback)

## 🎨 카테고리별 가이드

| 카테고리 | 가이드 |
|---------|--------|
| `health` | 건강, 웰빙, 생활 꿀팁과 관련된 이미지. 밝고 깔끔한 스타일, 전문적이고 신뢰감 있는 느낌. |
| `history` | 역사적 이미지. 역사적 사실과 배경을 정확하게 반영. 고전적이고 장엄한 스타일. |
| `wisdom` | 명언, 지혜, 철학과 관련된 이미지. 고요하고 깊이 있는 느낌, 영감을 주는 분위기. |
| `self_improvement` | 자기계발, 성장과 관련된 이미지. 동기부여되고 긍정적인 에너지. |
| `society` | 사회 트렌드, 최신 이슈와 관련된 이미지. 현대적이고 트렌디한 스타일. |
| `space` | 우주, 천문학과 관련된 이미지. 신비롭고 장엄한 우주의 느낌. |
| `fortune` | 사주, 운세와 관련된 이미지. 신비롭고 미스터리한 분위기. |

- 한국인: 한국 역사, 한국 문화, 한국인 인물, 한국 배경 등이 명확히 언급된 경우

- "산 풍경" → CHARACTER_TYPE: none, CHARACTER_DESCRIPTION: none

인물 정보를 파악할 수 없으면 "CHARACTER_TYPE: none, CHARACTER_DESCRIPTION: none"로 응답하세요.
```

```

```json
{
  "Authorization": "Token {replicateApiKey}",

```

  "width": 1360,

  "negative_prompt": "realistic human, detailed human skin, photograph, 3d render, blank white background, line-art only, text, letters, words, writing, labels, signs, watermark, typography, font, caption, subtitle, non-stickman, mixed style, detailed cartoon human, prince, princess, disney, pixar, anime, chibi, kawaii, big head, human body, human skin, realistic, 3d render, semi-realistic, detailed face, eyelashes, blush, nose, lips, hair, ears, detailed clothing folds, portrait, close-up, single character focus, bokeh, depth of field, watercolor, painterly, airbrush, soft shading",

#### google/imagen-4-fast
```json
{
  "prompt": "{finalPrompt}",

  "num_inference_steps": 4,
  "output_format": "png"
}
```

### 응답 처리 (폴링)

1. **초기 응답**: `status: "processing"` 또는 `status: "starting"`
2. **폴링**: 2초마다 상태 확인 (최대 120회, 약 4분)
3. **완료**: `status: "succeeded"` → `output`에서 이미지 URL 추출
4. **실패**: `status: "failed"` → 에러 메시지 반환

### 응답 형식
```json
{
  "id": "prediction_id",
  "status": "succeeded",
  "output": "https://replicate.delivery/..."
}
```

## ⚠️ 주의사항

1. **텍스트 제거**: 모든 이미지에서 텍스트, 글씨, 로고, 워터마크 등이 보이지 않도록 해야 합니다.
2. **캐릭터 일관성**: 실사화 스타일에서는 캐릭터의 모든 세부사항을 정확히 동일하게 유지해야 합니다.
