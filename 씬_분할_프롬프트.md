# 씬 분할 기능 - API 및 프롬프트 가이드

## 📋 개요
대본 생성 후 각 씬(Scene)을 장면(Shot) 단위로 분해하는 기능입니다. 시각적 일관성을 기준으로 하나의 씬을 최대 3개의 장면으로 나눕니다.

---

## 🔌 API 정보

### 기본 정보
- **API 엔드포인트**: `https://generativelanguage.googleapis.com/v1beta/models/gemini-3-flash-preview:generateContent`
- **HTTP 메서드**: `POST`
- **인증 방식**: API Key (쿼리 파라미터)
- **Content-Type**: `application/json`
- **사용 모델**: `gemini-3-flash-preview`

## 📋 시각적 일관성 기준

### 같은 장면으로 묶을 수 있는 경우

1. **배경이 동일한 경우**
   - 같은 장소, 공간, 환경
   - 예: 모두 궁궐 안에서 일어나는 일

2. **시대가 동일한 경우**
   - 같은 시대 배경
   - 예: 모두 조선시대

3. **인물 구성이 동일한 경우**
   - 같은 등장 인물
   - 예: 모두 세종대왕이 등장

4. **시점이 동일한 경우**
   - 같은 관찰 시점
   - 예: 모두 현재 진행형 서술

### 새로운 장면으로 분리해야 하는 경우

1. **배경이 바뀌는 경우**
   - 예: 궁궐 → 시골 마을

2. **시대가 바뀌는 경우**
   - 예: 조선시대 → 현대

3. **인물이 바뀌는 경우**
   - 예: 세종대왕 → 다른 인물

4. **시점이 바뀌는 경우**
   - 예: 현재 진행 → 회상

## 📋 구현 가이드

### 함수 시그니처

```typescript
async function decomposeScriptIntoScenes(
  script: string,
  apiKey?: string
): Promise<string>
```

### 처리 프로세스

1. **대본을 씬 단위로 분할**
   - 빈 줄 기준으로 분할
   - 빈 줄이 없으면 전체를 하나의 씬으로 처리

2. **긴 씬 분할**
   - 씬이 1000자 이상이면 문장 단위로 분할
   - 각 청크는 최대 1000자로 제한

3. **각 씬을 순차적으로 처리**
   - 각 씬을 AI에 전달하여 장면으로 분해
   - 최대 3회 재시도 (타임아웃 3분)
   - 실패 시 원본 씬을 기본 형식으로 사용

4. **결과 합치기**
   - 모든 씬의 결과를 합쳐서 반환

### 에러 처리

- **타임아웃**: 3분 초과 시 재시도 (최대 3회)
- **API 실패**: 재시도 후에도 실패하면 원본 씬을 기본 형식으로 사용
- **응답 형식 오류**: 원본 씬을 기본 형식으로 사용

## ⚠️ 주의사항

1. **문장 보존**: 입력 대본의 문장을 절대 변경하거나 요약하지 않습니다.
2. **장면 개수**: 최소 1개, 최대 3개로 제한합니다.
3. **문장 수**: 장면당 1~4문장 이내로 제한합니다.
4. **시각적 일관성**: 같은 화면으로 표현 가능한 단위로만 묶습니다.
5. **추가 문구 금지**: 장면 설명, 해설, 분석 문구를 추가하지 않습니다.

## 📋 씬 분할 전처리

### 빈 줄 기준 분할

```javascript
// 대본을 의미 단위(씬)로 나누기 (빈 줄 기준)
let scenes = script.split(/\n\s*\n/).filter((s: string) => s.trim().length > 0)

if (scenes.length === 0) {
  // 빈 줄이 없으면 전체를 하나의 씬으로 처리
  scenes = [script]
}
```

### 긴 씬 분할

```javascript
const MAX_SCENE_LENGTH = 1000 // 씬당 최대 길이

for (let idx = 0; idx < scenes.length; idx++) {
  const scene = scenes[idx]
  
  if (scene.length <= MAX_SCENE_LENGTH) {
    // 분할 불필요
    splitScenes.push(scene)
  } else {
    // 긴 씬을 문장 단위로 분할
    const sentences = scene.split(/([.!?。！？]\s*)/).filter((s: string) => s.trim().length > 0)
    let currentChunk = ""
    
    for (let i = 0; i < sentences.length; i++) {
      const sentence = sentences[i]
      if (currentChunk.length + sentence.length <= MAX_SCENE_LENGTH) {
        currentChunk += sentence
      } else {
        if (currentChunk.trim().length > 0) {
          splitScenes.push(currentChunk.trim())
        }
        currentChunk = sentence
      }
    }
    
    if (currentChunk.trim().length > 0) {
      splitScenes.push(currentChunk.trim())
    }
  }
}
```

## 📋 응답 검증

### 형식 검증

```javascript
// 최소한 "씬 N" 또는 "[장면 1]" 같은 패턴이 있는지 확인
const hasValidFormat = trimmedContent.match(/씬\s+\d+|\[장면\s+\d+\]/)

if (!hasValidFormat) {
  throw new Error(`씬 ${sceneNumber} 장면 분해 결과 형식이 올바르지 않습니다.`)
}
```
