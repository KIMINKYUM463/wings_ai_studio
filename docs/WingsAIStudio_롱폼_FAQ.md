## 🔑 API 키 관련

### Q1. 어떤 API 키가 필수인가요?
**A:** 다음 3개는 필수입니다:
- **OpenAI API 키**: 대본 개선, 제목/설명 생성, 자막 동기화 등
- **Gemini API 키**: 대본 생성, 콘텐츠 타입 추천
- **Replicate API 키**: 이미지 생성, 이미지→비디오 변환

나머지(ElevenLabs, TTSMaker, Supertone, YouTube Data API)는 사용하는 기능에 따라 선택입니다.

### Q2. API 키는 어디에 입력하나요?
**A:** 
1. WingsAIStudio 롱폼 페이지 접속
2. 우측 상단 **설정(⚙️)** 버튼 클릭
3. **API 키 설정** 탭 선택
4. 각 API 키 입력 후 **저장** 버튼 클릭

### Q3. API 키를 입력했는데도 "API 키가 필요합니다" 오류가 나요
**A:** 다음을 확인해주세요:
1. **저장 버튼을 클릭했는지** 확인 (입력만으로는 저장되지 않음)
2. **브라우저를 새로고침** 후 다시 시도
3. **브라우저 개발자 도구(F12) → Console**에서 에러 메시지 확인
4. **LocalStorage 확인**:
   ```javascript
   // 브라우저 콘솔에서 실행
   console.log(localStorage.getItem("openai_api_key"))
   ```

### Q4. API 키가 저장되지 않아요
**A:** 
1. **브라우저 쿠키/로컬스토리지 차단** 확인
2. **시크릿 모드가 아닌지** 확인
3. **다른 브라우저에서 시도**
4. **브라우저 캐시 삭제** 후 재시도

### Q5. API 키를 잃어버렸어요. 복구할 수 있나요?
**A:** 
- **OpenAI, Gemini, Replicate 등**: 각 서비스 대시보드에서 재발급 가능
- **저장된 키 확인**: 설정 페이지에서 "API 키를 메모장으로 저장" 기능 사용 (이미 저장된 경우)
- **백업**: 정기적으로 API 키를 안전한 곳에 백업하는 것을 권장합니다

### Q6. API 키 입력 시 "유효하지 않은 키" 오류가 나요
**A:** 
1. **공백 확인**: 키 앞뒤에 공백이 없는지 확인
2. **전체 키 복사**: 키가 잘리지 않았는지 확인
3. **키 형식 확인**: 각 API의 올바른 키 형식인지 확인
   - OpenAI: `sk-`로 시작
   - Gemini: 긴 문자열
   - Replicate: `r8_`로 시작

## 📝 대본 기획 관련

### Q7. 대본 기획안이 생성되지 않아요
**A:** 다음을 확인해주세요:
1. **Gemini API 키**가 입력되어 있는지 확인
2. **콘텐츠 타입**이 선택되어 있는지 확인 (Type A, B, C, D 또는 커스텀 타입 중 하나 필수)
3. **커스텀 타입 선택 시**: 원하는 대본 구조 설명을 입력했는지 확인
4. **주제와 카테고리**가 선택되어 있는지 확인
5. **브라우저 콘솔**에서 에러 메시지 확인
6. **Gemini API 할당량** 확인 (일일 한도 초과 가능)

### Q8. "콘텐츠 타입을 선택해주세요" 오류가 나요
**A:** 
- 대본 기획 기능은 반드시 **콘텐츠 타입(Type A, B, C, D 또는 커스텀)**을 선택해야 합니다
- 주제 선택 후 **콘텐츠 타입 선택** 단계에서 타입을 선택해주세요
- **커스텀 타입** 선택 시에는 원하는 대본 구조 설명도 함께 입력해야 합니다

### Q9. 대본 기획 생성이 너무 느려요
**A:** 
- Gemini API는 기획안 길이에 따라 시간이 걸릴 수 있습니다
- 긴 영상(10분 이상)의 기획안은 30초~1분 정도 소요될 수 있습니다
- 네트워크 상태를 확인해주세요

3. **브라우저 콘솔**에서 에러 메시지 확인

### Q11. 대본 생성이 너무 느려요

## 🖼️ 이미지 생성 관련

### Q14. 이미지가 생성되지 않아요
**A:** 다음을 확인해주세요:
1. **Replicate API 키**가 입력되어 있는지 확인
2. **OpenAI API 키**가 입력되어 있는지 확인 (이미지 프롬프트 생성에 필요)
3. **Replicate 계정에 크레딧**이 충분한지 확인
4. **이미지 모델**이 올바르게 선택되어 있는지 확인

### Q15. 이미지 생성이 실패해요
**A:** 
- **Replicate API 오류**: Replicate 대시보드에서 서비스 상태 확인
- **크레딧 부족**: Replicate 계정에 크레딧 충전 필요
- **프롬프트 문제**: 부적절한 프롬프트는 거부될 수 있음

### Q16. 이미지 생성이 너무 느려요
**A:** 
- 이미지 생성은 모델에 따라 10-30초 정도 소요됩니다
- **이미지 모델 변경**: 더 빠른 모델 선택 (예: `flux-schnell`)
- **네트워크 상태** 확인

### Q17. 생성된 이미지가 마음에 들지 않아요
**A:** 
- **이미지 재생성** 버튼 사용
- **이미지 스타일** 변경
- **커스텀 이미지 프롬프트** 입력

### Q18. 이미지→비디오 변환이 실패해요
**A:** 
- **Replicate API 키** 확인
- **Replicate 크레딧** 확인
- **이미지가 먼저 생성**되어 있어야 변환 가능

## 🎤 음성 합성(TTS) 관련

### Q19. 음성이 생성되지 않아요
**A:** 
- 사용하는 **TTS 서비스의 API 키**가 입력되어 있는지 확인:
  - **TTSMaker 음성**: TTSMaker API 키 필요
  - **ElevenLabs 음성**: ElevenLabs API 키 필요
  - **Supertone 음성**: Supertone API 키 필요
- **대본이 생성**되어 있어야 음성 생성 가능

### Q20. TTSMaker 음성이 작동하지 않아요
**A:** 
- **TTSMaker API 키**가 설정 페이지에 입력되어 있는지 확인
- TTSMaker 웹사이트에서 서비스 상태 확인
- 다른 TTS 서비스(ElevenLabs, Supertone)로 변경 시도

### Q21. ElevenLabs 음성이 작동하지 않아요
**A:** 
- **ElevenLabs API 키** 확인
- ElevenLabs 계정의 **할당량/크레딧** 확인
- **음성 ID**가 올바른지 확인

### Q22. Supertone 음성이 작동하지 않아요
**A:** 
- **Supertone API 키** 확인
- **음성 ID와 스타일**이 올바르게 선택되었는지 확인
- Supertone 대시보드에서 서비스 상태 확인

### Q23. 음성 품질이 좋지 않아요
**A:** 
- **다른 TTS 서비스**로 변경 시도 (ElevenLabs가 일반적으로 품질이 좋음)
- **음성 스타일** 변경
- **대본 발음 기호** 확인 및 수정

## 📺 자막 관련

### Q24. 자막이 정확하지 않아요
**A:** 
- **OpenAI API 키**가 입력되어 있어야 정확한 자막 동기화 가능
- API 키가 없으면 대체 방식(텍스트 길이 비율)을 사용하여 정확도가 낮을 수 있음
- **자막 재생성** 시도

### Q25. 자막이 음성과 맞지 않아요
**A:** 
- **자막 동기화** 기능 사용 (OpenAI API 키 필요)
- **수동으로 자막 시간 조정** 가능
- **Whisper API**를 사용하면 더 정확한 동기화 가능

### Q26. 자막이 표시되지 않아요
**A:** 
- **자막 표시 옵션**이 켜져 있는지 확인
- **자막이 생성**되었는지 확인
- **비디오 재생** 중인지 확인 (자막은 재생 중에만 표시)

## 🎬 YouTube 업로드 관련

### Q27. YouTube 업로드가 실패해요
**A:** 
1. **YouTube 계정 연결** 확인:
   - 설정 → YouTube 연결
   - "YouTube 연결" 버튼 클릭하여 인증
2. **토큰 만료** 확인:
   - "YouTube 토큰이 만료되었습니다" 오류 시 재연결 필요
3. **비디오 파일** 확인:
   - 비디오가 정상적으로 생성되었는지 확인

### Q28. YouTube 예약 업로드가 작동하지 않아요
**A:** 
- **YouTube 계정 연결** 확인
- **예약 날짜/시간**이 올바른지 확인 (과거 시간 불가)
- **브라우저를 닫지 말고** 예약 시간까지 대기 (또는 서버에서 처리되는 경우 서버 상태 확인)

### Q29. YouTube 연결이 끊어졌어요
**A:** 
- YouTube OAuth 토큰은 일정 시간 후 만료됩니다
- 설정에서 **YouTube 재연결** 필요
- 정기적으로 연결 상태를 확인하는 것을 권장합니다

## 🔧 기타 문제

### Q30. 전체 기능이 작동하지 않아요
**A:** 
1. **필수 API 키 3개** 확인:
   - OpenAI API 키
   - Gemini API 키
   - Replicate API 키
2. **브라우저 새로고침** (F5)
3. **브라우저 캐시 삭제**
4. **다른 브라우저에서 시도**

### Q31. 특정 기능만 작동하지 않아요
**A:** 
- 해당 기능에 필요한 **API 키** 확인 (조교 메뉴얼 참고)
- **브라우저 콘솔(F12)**에서 에러 메시지 확인
- **API 제공업체 상태 페이지** 확인:
  - OpenAI: https://status.openai.com
  - Google Cloud: https://status.cloud.google.com
  - Replicate: https://status.replicate.com

### Q32. "네트워크 오류"가 발생해요
**A:** 
- **인터넷 연결** 확인
- **방화벽/보안 프로그램** 확인
- **VPN 사용 중**이면 해제 후 시도
- **다른 네트워크**에서 시도

### Q33. 작업 중 페이지가 멈춰요
**A:** 
- **긴 작업**은 시간이 걸릴 수 있습니다 (이미지 생성, 비디오 렌더링 등)
- **브라우저 콘솔**에서 진행 상황 확인
- **페이지를 닫지 말고** 완료될 때까지 대기
- 5분 이상 멈춰있으면 **새로고침** 후 재시도

### Q34. 생성된 비디오가 재생되지 않아요
**A:** 
- **비디오 파일**이 정상적으로 생성되었는지 확인
- **브라우저 호환성** 확인 (Chrome, Edge 권장)
- **다른 브라우저**에서 시도
- **비디오 다운로드** 후 로컬 플레이어로 재생 시도

### Q35. 저장된 작업이 사라졌어요
**A:** 
- WingsAIStudio는 **브라우저 LocalStorage**에 데이터를 저장합니다
- **브라우저 데이터 삭제** 시 작업이 사라질 수 있습니다
- **정기적으로 다운로드**하여 백업하는 것을 권장합니다

### Q36. API 사용량이 너무 많아요
**A:** 
- 각 API 제공업체의 **대시보드**에서 사용량 확인
- **할당량/크레딧** 확인
- 필요시 **플랜 업그레이드** 또는 **크레딧 충전**

### Q37. 비용이 얼마나 드나요?
**A:** 
- **OpenAI**: 사용량 기반 (토큰 수)
- **Gemini**: 무료 할당량 제공, 초과 시 유료
- **Replicate**: 크레딧 기반 (이미지당 약 $0.002-0.01)
- **TTS 서비스**: 서비스별 상이 (ElevenLabs는 문자 수 기반)
- 각 서비스의 **가격 정책**을 확인하세요
