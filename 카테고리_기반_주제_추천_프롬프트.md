# 롱폼 주제 15개 생성 - 카테고리 기반 모드

## 📋 개요
선택한 카테고리와 키워드를 기반으로 유튜브 롱폼 비디오 주제 15개를 자동으로 생성하는 기능입니다.

---

## 🔌 API 정보

### 기본 정보
- **API 엔드포인트**: `https://api.openai.com/v1/chat/completions`
- **HTTP 메서드**: `POST`
- **인증 방식**: Bearer Token (Authorization 헤더)
- **Content-Type**: `application/json`

### 사용 모델
- **모델명**: `gpt-4o-mini`
- **max_tokens**: `1500`
- **temperature**: `0.9`

### API 요청 헤더
```json
{
  "Authorization": "Bearer {YOUR_API_KEY}",
  "Content-Type": "application/json"
}
```

### 변수 설명

- **`{categoryPrompt}`**: 카테고리 설명 (아래 카테고리 목록 참고)
- **`{keywordsText}`**: 
  - 키워드가 있는 경우: `\n키워드: {keywords}` 형식으로 추가

## 🔄 응답 처리

### 응답 형식
API 응답은 다음과 같은 구조입니다:

```json
{
  "choices": [
    {
      "message": {
        "content": "1. 주제명1\n2. 주제명2\n3. 주제명3\n..."
      }
    }
  ]
}
```

### 파싱 로직

1. **응답 추출**: `data.choices[0].message.content`에서 텍스트 추출
2. **줄 단위 분리**: `\n`으로 분리하고 빈 줄 제거
3. **주제 추출**: 정규식으로 번호 형식 매칭
   - 지원 형식: `1.`, `1)`, `- 1.`, `* 1.` 등
   - 정규식: `/^[\d\-]+[\.\)]\s*(.+)$/` 또는 `/^[\-\*]\s*(.+)$/`
4. **필터링**: 오류 메시지 제거
   - 제외 키워드: "카테고리가", "죄송하지만", "제공해"
5. **개수 제한**: 최대 15개까지 수집

### 파싱 코드 예시 (JavaScript)

```javascript
const data = await response.json();
const content = data.choices[0]?.message?.content;

if (!content) {
  throw new Error("API 응답에서 내용을 찾을 수 없습니다.");
}

const lines = content.split("\n").filter((line) => line.trim());
const topics = [];

for (const line of lines) {
  // 다양한 번호 형식 지원: "1.", "1)", "1. ", "- 1." 등
  const match = line.match(/^[\d\-]+[\.\)]\s*(.+)$/) || line.match(/^[\-\*]\s*(.+)$/);
  if (match && match[1]) {
    const topic = match[1].trim();
    // 오류 메시지가 아닌 실제 주제인지 확인
    if (
      topic &&
      topic.length > 0 &&
      !topic.includes("카테고리가") &&
      !topic.includes("죄송하지만") &&
      !topic.includes("제공해")
    ) {
      topics.push(topic);
    }
  }
  if (topics.length >= 15) break;
}

if (topics.length === 0) {
  throw new Error(`주제를 생성할 수 없습니다. GPT API 응답: ${content.substring(0, 200)}`);
}

return topics;
```
